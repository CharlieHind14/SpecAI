# SpecAI
Spec AI is my custom AI system.

It is self taught and the more it is used the smarter it gets - this works because of its read and write databases:
- Preprogrammed database - Basic phrases and information about itself ( 'Hi', 'how are you?', 'who made you?', etc)
- learnt database -  If the AI does not currently in any of its databases have a response for the users input it asks for an ideal response ( this will be replaced in future by a automatic websearch ).
- user database - For the First use of the AI it asks for you name and favourite colour ( This is to be extended with favourite bands and books etc )
This function is to be extended in future as currently every time you restart the program all new memory is lost :(

Current Features:
- Basic Chat functions
- Read and Write databases
